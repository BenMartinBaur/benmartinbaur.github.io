# Writing

I've never been more excited about **technology**. I am not a developer, never been an expert in writing powershell or any other type of script. For the first time in my career, anything I want to build, write or create in the software world feels fundamentally within my grasp.

Not because I've suddenly become a better engineer. Not because I am out of a suddon a graphic designer. Not because tools have gotten marginally more convenient. Something has shifted at a deeper level—something about the relationship between what I can imagine and what I can create.

Today I built this entire blog. From scratch. With **zero coding skills**. How? **GitHub Copilot**. In a single conversation, GitHub Copilot helped me create a complete blog platform.
