# Innovation Hub — Brand Spec for Word Recaps

Distilled from **Innovation Hub Creative Expression Guide** (January 2026) for use
when building the customer workshop recap Word document. Source PDF and assets live in
SharePoint: **MTC team → Shared Documents → Identity and templates**
(https://microsoft.sharepoint.com/teams/MTC/Shared%20Documents/Identity%20and%20templates).

> Note: that folder holds PowerPoint templates (.potx), the design PDF, and a photography
> brief — there is **no Word (.dotx) template**. Build the .docx to the spec below.

## Naming (strict)
- **First mention:** "Microsoft Innovation Hub". **After that:** "Innovation Hub" or "Hub".
- **Always singular** — never "Innovation Hubs". **Never** use the initialism "MIH".
- Innovation Hub is a **program identity, not a brand** — it has **no logo**. Roll up to the
  standard **Microsoft logo** only. Do not pair the program name with the Microsoft squares
  logo, and do not place it directly beside the Microsoft logo ("double Microsoft").

## Color palette (hex)
**Primary accents**
- Magenta `#c03bc4` — the hero accent (use most for emphasis)
- Blue `#0078d4`
- Light Orange `#ffa38b` — *light warm gradient only*
- Orange `#ff5c39` — *light warm gradient only*

**Neutrals**
- Dark Blue (Expanded Blue 1) `#152237` — primary dark; default for headings/title
- Warm White `#fff8f3` — default page background
- Blue Black `#091f2c` — body text and fine detail
- Warm Light Gray `#e8e6df` — subtle fills / callout blocks
- Mid Gray `#d7d2cb` — dividers / secondary lines
- Pure White `#ffffff`

**Charts/graphs** use the expanded palette and skip every other value for separation, e.g.
Magenta `#c03bc4`, Blue `#0078d4`, Light Blue `#8dc8e8`, Orange `#ff5c39`, Warm Gray `#8c8279`.

## Typography
Brand fonts are **Segoe Sans Display** (headings) and **Segoe Serif Display** (quotes/callouts).
These are licensed and usually absent from the document container — use these fallbacks:
- **Headings / title:** Segoe UI Semibold (brand: Segoe Sans Display Semibold), color Dark Blue `#152237`.
- **Body:** Segoe UI Regular ~10–11 pt, color Blue Black `#091f2c`.
- **Pull quotes / testimonials & location callouts:** Georgia *italic* (brand: Segoe Serif Display Semilight).
- Pick ~3 sizes with clear contrast (skip a step). Practical ramp: body 10–11, H3 13, H2 20–22, H1/title 28–32.

## Gradient (use sparingly, optional in Word)
- Light warm gradient = Magenta `#c03bc4` → Light Orange `#ffa38b` → Orange `#ff5c39`, linear 40°.
- **Headlines only**, on solid neutral backgrounds. Never on body copy, never over images/illustration,
  never on the "Microsoft Innovation Hub" rubric, never below 16 pt bold. Only one gradient type per page.
- In Word, gradient text is impractical — **default the title to solid Magenta `#c03bc4` or Dark Blue `#152237`**.

## Type don'ts
- Don't set standard headlines or body copy in the serif face.
- Don't set number callouts in the sans face (numerals → Segoe Serif Display Regular / Georgia).
- Don't mix more than one serif typestyle per layout.

## Recap document layout (apply this)
- **Page:** Warm White `#fff8f3` background; generous margins.
- **Title block:** "Customer Workshop Recap" in Dark Blue `#152237` (or Magenta accent), with a thin
  Magenta `#c03bc4` rule beneath. Subtitle line: customer · date.
- **Section headings:** Segoe UI Semibold, Dark Blue `#152237`.
- **Tables** (attendees, action items): header row filled Dark Blue `#152237` with Warm White text;
  body rows alternate White / Warm Light Gray `#e8e6df`.
- **Pull quote** (optional customer quote): Georgia italic, Blue Black `#091f2c`, in a Warm Light Gray block.
- **Footer:** "Microsoft Innovation Hub" wordmark text + page number; Microsoft logo roll-up only.
