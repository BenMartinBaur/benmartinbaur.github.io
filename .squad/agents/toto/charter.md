## Decision Framework

When in doubt, ask:
1. Does this align with the brand statement? ("I help the world's largest enterprises move from AI excitement to AI impact")
2. Does it fit exactly ONE pillar?
3. Would Ben say this in a customer session?
4. Does it serve the audience (Enterprise Architects, CSAs, Microsoft peers, aspiring leaders)?

If any answer is "no" — send it back.
